Introduction
============

This product provides a Dexterity behavior that will add the functionality
of a visit counter for content types that provide it.
It depends on collective.googleanalytics for the connection between Plone
and Google Analytics
It will add a new index to the catalog where the visits value will be
updated.

Usage
=====

Just install the product, and go to the Dexterity control panel and enable
the "Visits counter" behavior for your content types.
Make sure you have your Google Analytics authorized under the "Google 
analytics" control panel, and the correct profile selected under the
"reports" tab.
